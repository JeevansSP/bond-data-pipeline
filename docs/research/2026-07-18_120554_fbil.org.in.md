# FBIL — fbil.org.in (Financial Benchmarks India Ltd)

_Explored 2026-07-18. FBIL is the RBI-recognized administrator of Indian financial benchmarks
(the official daily valuation curves the whole market marks to)._

## Why it matters for this project
FBIL publishes **daily per-security valuation PRICE + YIELD** for G-Secs and SDLs, plus T-Bill
tenor rates and yield curves. For a hold-to-maturity **ladder backtest**, this is the cleanest
**historical price/yield** source for the sovereign universe — authoritative, daily, free.

## API — no auth, clean JSON
Base: `https://www.fbil.org.in/wasdm/<product>/fetch?authenticated=false`
- **GOTCHA:** send `Accept: application/json, text/plain, */*` — without it the server returns
  **HTTP 406** (empty). Also send `Referer: https://www.fbil.org.in/`. No cookies/auth needed for GET.
- History (money-market products): `.../<product>/fetchcommongraph/3M` or `/fetchgraphdata/3M`
  (3-month graph series; other windows likely by swapping `3M`).

### Product families
**A) Direct-rate products** — `/fetch` returns the data inline:
| product | content | row schema |
|---|---|---|
| `tbill` | T-Bill implied rates by tenor | `{processRunDate, displayTime, tenorName ("7 Days"…25 tenors), rate, comments}` |
| `ovnmibor`,`termmibor`,`miborois` | MIBOR (overnight/term/OIS) | date, tenor, time, rate |
| `mror`,`sorr` | market repo / secured overnight rupee rate | date, tenor, time, rate |
| `cd` | Certificate of Deposit rates | date, time, tenor, rate |
| `refrates` | USD/INR (and majors) reference rate | date, rate |
| `fwdpremia`,`mifor`,`modifiedmifor`,`adjustedmifor`,`mmswaprates`,`fcrupee` | FX/derivatives benchmarks | date/tenor/rate |

**B) File-archive products** — `/fetch` returns a **date index**, the data itself is a downloadable
daily Excel/CSV (per-security):
| product | archive title | contains |
|---|---|---|
| `gsec` | "FBIL GOI Prices including Par Yield, Special and Note on FRB and IIB" | **per-ISIN G-Sec price + par yield** (incl. floating-rate & inflation-indexed) |
| `sdl` | "FBIL SDL /SGS Prices including UDAY and Tenor Average YTM" | **per-ISIN SDL price + YTM**, tenor-average YTM |
| `sdlzcyc` | "FBIL SDL ZCYC" | SDL zero-coupon yield curve params |
| `strips` | G-Sec STRIPS valuation | per-STRIP price/yield |
  (`/<product>/fetch` returns e.g. `[{"processRunDate":"2026-07-10","archive":"…"}, …]` — recent dates.)

## Open loose end (resolve at implementation)
The exact **file-download URL** for `gsec`/`sdl` daily valuation files isn't yet confirmed — the
SPA renders "Download Excel / Download CSV" buttons whose click fires JS (a programmatic click
didn't emit a `/wasdm` XHR; guessed paths `downloadfile`/`fetcharchivefile`/`archive` all 404).
Some FBIL files are served from `https://www.fbil.org.in/uploads/<name>_<hash>.xlsx` (seen for FRB
polling / ARR sheets). **To pin it:** in a browser, click the G-Sec date row and read the actual
download request in the network tab. Historical depth of the archive (how far back dates go) also
TBD — `/fetch` showed only the latest 2 dates; FBIL retains history, likely via a date param.

## Quality
- **Strong / authoritative** — these are *the* official valuations; daily (published ~evening IST).
- Sovereign coverage (G-Sec, SDL, T-Bill) is complete and per-security. **No corporate-bond
  per-security valuation here** (that's FIMMDA / exchanges).
- Clean JSON + Excel; trivial to script once the file endpoint is pinned.

## Fit for the pipeline
- **Pillar: historical price/yield (sovereign).** Daily snapshot of `gsec`+`sdl` files → per-ISIN
  price/YTM time series = the ladder's mark-to-market + entry-yield history. `tbill` + ZCYC for the
  curve / reinvestment assumptions. See [[project-indian-bond-research]].
