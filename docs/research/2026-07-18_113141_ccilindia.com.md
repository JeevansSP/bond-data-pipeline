**CSV schema** (verified, ~872 KB for 2 days):
`Trade , Trade Time , ISIN , Security Description , Face Value , Trade Price , YTM/Yield , Trade Indicator`
e.g. `17-07-2026,16:59:59,IN0020260025,06.94 GOVT. STOCK 2036,50000000.000,101.1150,6.7806,NRML`
and SDL `03-06-2025,...,IN2220190127,06.97 MAHARASHTRA SDL 2028,...`. Date `DD-MM-YYYY`; Face Value =
traded amount (₹, not ISIN face); Trade Indicator NRML. Confirmed working for 2025 and 2026 ranges.

**The "encryption" (fully reversible):** `hidFrom/hidTo = encryptDetails(date)` where
`encryptDetails` = **AES-128-ECB, PKCS7**, key = base64 `bXVzdGJlMTZieXRlc2tleQ==` = **`mustbe16byteskey`**
(16 bytes), output base64. Python (`cryptography`) reproduces the site's ciphertext byte-for-byte:
```python
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes
from cryptography.hazmat.primitives import padding
import base64
_KEY = b"mustbe16byteskey"
def encrypt_date(s: str) -> str:
    pad = padding.PKCS7(128).padder(); data = pad.update(s.encode()) + pad.finalize()
    enc = Cipher(algorithms.AES(_KEY), modes.ECB()).encryptor()
    return base64.b64encode(enc.update(data) + enc.finalize()).decode()
# encrypt_date("16/07/2026") == "6AUyCd+B1T3jyWBfbj3Kng==" (matches browser)
```

## Access mechanics (how to scrape)
- Platform: **Liferay DXP portal** behind **Akamai** (`_abck`, `bm_sz`, `ak_bmsc`, `bm_sv` cookies +
  `JSESSIONID`). Load the page first to mint cookies, then reuse them.
- Data isn't a REST API — it's **Liferay portlet resource calls**:
  ```
  POST https://www.ccilindia.com/<page>?p_p_id=<portletId>&p_p_lifecycle=2
       &p_p_state=normal&p_p_resource_id=<resourceId>&p_p_cacheability=cacheLevelPage
  header: X-Requested-With: XMLHttpRequest
  ```
  e.g. individual-trades portlet `com_ccil_individual_trades_CcilIndividualTradesMVCPortlet_INSTANCE_bxkl`,
  `resource_id=ticker` → `{"resultMarketOpenClose":"N"}` when closed. Actual trade rows come from the
  portlet's data `resource_id` (discover live during market hours; each page has its own portlet id).
- **Live feeds are intraday-only** — closed on weekends/holidays (checked a Saturday → no ticks).
  For history, use the settlement stats, Rakshitra/Factbook publications, and archives.
- Rate-limit risk high (Akamai) → throttle, single session, respect market hours.
