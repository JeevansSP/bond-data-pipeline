# NSE — nseindia.com (Corporate Bonds Traded on Exchange)

_Explored 2026-07-18. Page: `/market-data/debt-market-reporting-corporate-bonds-traded-on-exchange`._

## What it is
NSE's **debt-market reporting** for corporate bonds — this IS trade/transaction data (unlike
BondCentral/CDSL). Powered by a JSON API discovered via network sniffing.

## The trade API
```
GET https://www.nseindia.com/api/liveCorp-bonds?index=otctrades_listed&marketType=CBM
```
**Response envelope** (verified round 2): `{"data":[…], "timestamp":"17-Jul-2026 17:09", "descriptors":[…], "marketStatus":{"market":"CBM", …}}`. `data` is the trade rows; `timestamp` = last session time; `marketStatus` tells you if the market is open.

Per-row schema (one row per traded ISIN):
| field | meaning |
|---|---|
| `descriptor` | security name (e.g. "THE TATA POWER COMPANY LIMITED 7.50 NCD 14JL31 FVRS1LAC") |
| `isin` | ISIN |
| `ltp` | last traded **price** |
| `lty` | last traded **yield** |
| `noOfTrades` | number of trades |
| `tradeValue` | traded value (₹ thousands/lakhs — verify units) |
| `wap` | weighted-average price |
| `way` | weighted-average yield |

- **`index` values (all confirmed round 2, `marketType=CBM` = Corporate Bond Market):**
  - `otctrades_listed` — OTC trades in listed bonds (**main feed; 153 rows on the last, 17-Jul, session**)
  - `otctrades_unlisted` — OTC trades in unlisted bonds (valid; 0 rows at check time)
  - `exchtrades_listed` — exchange-traded, listed (valid; 0 rows at check time)
  - `exchtrades_unlisted` — exchange-traded, unlisted (valid; 0 rows at check time)
  All four return 200 with the same envelope; empty ones still carry `timestamp`/`marketStatus`.
- Related endpoints on the same page: `/json/corporate-bonds.json` (static), plus standard NSE
  `/api/marketStatus`, `/api/eventnotification`, `/api/getNotes20`.

## Access / anti-bot (IMPORTANT)
- Site is behind **Akamai**. Endpoints require valid cookies (`_abck`, `bm_sz`, `ak_bmsc`,
  `bm_sv`) that are minted only after loading the HTML page in a real browser session.
- Plain `curl`/WebFetch to the API → blocked. **Scrape flow:** (1) GET the page in a browser/session
  to obtain cookies, (2) reuse that cookie jar + a browser `User-Agent` + `Referer: <page url>`
  for the `/api/liveCorp-bonds` call. Cookies rotate (~hours) → refresh periodically.
- **Rate-limit risk is high** — throttle hard, single-threaded, random delays. Repeated rapid hits
  get the session flagged.
- Data is intraday/live (empty when market closed). For history, use end-of-day trade reports /
  archives rather than polling this endpoint.

## Fit for research
- Primary **exchange corporate-bond trade** source: price, yield, #trades, value, WAP/WAY per ISIN.
- Pair with BSE (see bondcentral doc's BSE bridge) for cross-exchange trade coverage.
