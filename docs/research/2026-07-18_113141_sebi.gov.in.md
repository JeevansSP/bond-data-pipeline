# SEBI — sebi.gov.in (Corporate Bond Public Issue Data)

_Explored 2026-07-18. Page: `/statistics/corporate-bonds/publicissuedata.html`._

## What it is
The regulator's record of **public issues of corporate bonds / NCDs** (retail-facing prospectus
issues). **Primary-market issuance only** — no private placements (the bulk of the market), no
secondary trades. Authoritative and clean for the public-issue slice.

## Structure & schema
- **~18 annual HTML tables**, one per financial year, **FY2008-09 → FY2026-27** (earliest entry
  02-Feb-09 Tata Capital; latest 23-Mar-26 Muthoot Fincorp).
- **Columns (consistent across years):** S.No · Name of company · Issue opened on · Issue closed on ·
  Base issue size (₹ Cr) · Final Issue size (₹ Cr)*.
  - *Footnote:* Final = Base + oversubscription per prospectus; "As per Initial Post Issue Report".
- **No file downloads** (no PDF/Excel/CSV) — data lives inline in the HTML tables. Scrape = parse the
  tables directly.

## Scraping notes
- Static server-rendered HTML, **no JS, no auth**. BUT (round-2): raw scripted `curl` intermittently
  gets **HTTP 530** (origin block / edge throttle); the same URL loads fine via WebFetch / a real
  browser. Use a browser-like client (proper UA, maybe cookies) and don't hammer.
- Year tables are all on the one page (year-wise sections). Parse each `<table>`; map rows to the
  6 columns above. Trivial, low-risk scrape.
- Good for a clean **public-issue calendar** (who raised how much, when) — join to ISIN-level sources
  by company name + dates (SEBI here doesn't carry ISINs).

## Related SEBI data (not this page, worth noting)
- SEBI Bond/Debenture statistics section (`/statistics/corporate-bonds/…`) also historically hosts
  private-placement and trading summaries; the wider corporate-bond stats index is worth a follow-up
  crawl for private-placement and secondary-market aggregate tables.

## Fit for research
- The **public-issue primary-market** view: issue windows and sizes since 2008. Narrow slice
  (public issues only), but authoritative and easiest of the four to ingest.
